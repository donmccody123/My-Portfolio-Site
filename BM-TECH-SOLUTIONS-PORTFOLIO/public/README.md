# Public Assets Folder

Place your logo file here as `logo.png` for it to appear in the navigation bar and hero section.

The logo should be:
- Named: `logo.png`
- Format: PNG (with transparent background recommended)
- Recommended size: At least 200x200px for best quality

The site will automatically use your logo when placed in this folder. If the logo file is not found, the site will fall back to displaying the brand name only.
